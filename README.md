# TermSnap

AI 기반 터미널 도우미 — "PuTTY를 더 편하게"

## 주요 기능

- **SSH 서버 세션** — 원격 서버 연결, AI 명령어 생성, SFTP 파일 전송
- **로컬 터미널 세션** — PowerShell, CMD, WSL, Git Bash 통합
- **AI 명령어 변환** — 자연어로 Linux 명령어 생성 (Gemini, OpenAI, Claude, Grok 지원)
- **Q&A 벡터 검색** — 자주 쓰는 명령어를 자동 저장하고 재사용
- **명령어 스니펫** — 저장된 명령어를 빠르게 실행
- **SFTP 파일 전송** — 드래그 앤 드롭으로 파일 전송
- **프로젝트 서브탭** — 여러 터미널을 하나의 프로젝트로 관리
